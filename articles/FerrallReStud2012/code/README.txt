FILES
Inputs
    SSPAppendixB.pdf        Empirical Moments as PDF
    empmom44.raw            Empirical Moments as Data Input
    sspmkzzz.ado            Stata program to generate empmom44.raw from micro data sets (not available)

    ssp44g.pas              GNU Pascal program that solves and estimates the model
                            Optimization carried by fmopt package (available from the author)
    ssp44m.start            Estimated parameters

    opt_XX.mat              Second stage weighting matrix for empirical group XX (large text files)

    social_experiments.pdf  A unpublish(ed/able) paper that describes the framework used

    Other files are text input or include files for ssp44g.pas

Outputs
    gphmoments44.dta        Empirical and Model Predictions (including experiments)
    grphssp.ado             Stata program to create graphs used in the pape
