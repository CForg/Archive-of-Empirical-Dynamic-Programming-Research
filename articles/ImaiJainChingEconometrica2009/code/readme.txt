FORTRAN COMPUTER CODE FOR THE IMPLEMENTATION OF THE BAYESIAN 
METHODS IN THE PAPER "Bayesian Estimation of Dynamic Discrete 
Choice Models," BY S. IMAI, Imai, N. JAIN and A. CHING

Files for the basic model
bayes.f: estimation program
dprog.f: simulation program
input.dat: parameter file

Files for the random effects models
bayesr.f: estimation program
dprogr.f: simulation program
inputr.dat: parameter file

Files for the continuous state space deterministic transition models.
bayesc.f: estimation program
dprogc.f: simulation program
inputc.dat: parameter file
